# Угадай Число

Шаблон для создания игры на JavaScript.

## Файлы проекта

- **`src/game.js`** — открой этот файл и напиши логику игры внутри трёх функций (четвёртая уже готова)
- **`src/game.solution.js`** — готовое решение (если застрял — загляни сюда)
- **`tests/game.test.js`** — тесты, которые проверят правильность твоего кода
- **`package.json`** — настройки проекта

## Что нужно сделать

Открой файл `src/game.js`. Там уже есть три функции, но они пустые. Реализуй их:

1. **`generateSecretNumber(min, max)`** — генерирует случайное целое число от `min` до `max` включительно
2. **`checkGuess(guess, secret)`** — сравнивает догадку игрока с секретным числом и возвращает подсказку
3. **`playRound(guess)`** — проводит один раунд: генерирует секретное число (по умолчанию от 1 до 100) и проверяет догадку

Функция **`playInteractiveGame(min, max)`** уже реализована — она запускает полноценную игру в консоли, где компьютер загадывает число, а ты вводишь догадки до тех пор, пока не угадаешь.

После каждого изменения запускай тесты командой:

```bash
npm test
```

Когда всё готово, поиграй в консоли:

```bash
npm start
```

Тесты подскажут, что работает, а что ещё нужно доделать.

## Правила игры

- Компьютер загадывает случайное число (например, от 1 до 100)
- Игрок пытается его угадать
- Если догадка меньше загаданного — верни `'too-low'`
- Если догадка больше загаданного — верни `'too-high'`
- Если догадка равна загаданному — верни `'correct'`
