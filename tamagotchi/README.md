# Тамагочи

Шаблон для создания текстового квеста на JavaScript.  
Здесь ты поработаешь с **объектами** — будешь создавать питомца, менять его свойства и следить за состоянием.

## Файлы проекта

- **`src/game.js`** — открой этот файл и напиши логику игры внутри функций (интерактивный режим уже готов)
- **`src/game.solution.js`** — готовое решение (если застрял — загляни сюда)
- **`tests/game.test.js`** — тесты, которые проверят правильность твоего кода
- **`package.json`** — настройки проекта

## Что нужно сделать

Открой файл `src/game.js`. Там уже есть функции, но три из них пустые. Реализуй их:

1. **`createPet(name)`** — создаёт и возвращает объект питомца со стартовыми статами
2. **`feed(pet)`** — кормит питомца: уменьшает голод, немного лечит
3. **`play(pet)`** — играет с питомцем: повышает счастье, но тратит энергию и повышает голод
4. **`sleep(pet)`** — укладывает спать: восстанавливает энергию, но немного повышает голод
5. **`checkStatus(pet)`** — возвращает красивую строку с текущими статами

Функция **`playInteractiveGame()`** уже реализована — она запускает полноценную игру в консоли, где ты выбираешь действия и следишь за питомцем.

После каждого изменения запускай тесты командой:

```bash
npm test
```

Когда всё готово, поиграй в консоли:

```bash
npm start
```

## Правила игры

- У питомца есть статы: `health` (здоровье), `hunger` (голод), `happiness` (счастье), `energy` (энергия), `gold` (золото), `dead` (жив ли)
- **Кормить** — уменьшает голод, немного лечит
- **Играть** — повышает счастье, но тратит энергию и повышает голод. Если голод достигает максимума — питомец теряет здоровье и может умереть!
- **Спать** — восстанавливает энергию, но немного повышает голод
- **Статус** — показывает все текущие статы
