# Установка окружения для разработки на Windows

## Подготовка к работе с репозиторием

Перед началом работы необходимо сделать **fork** данного репозитория в свой аккаунт GitHub и прислать ссылку на него.

### Как сделать fork:

1. Убедитесь, что вы авторизованы на [GitHub](https://github.com/)
2. Откройте страницу данного репозитория
3. Нажмите кнопку **Fork** в правом верхнем углу страницы
4. Выберите свой аккаунт — репозиторий будет скопирован в ваш профиль
5. Скопируйте ссылку на ваш fork (например, `https://github.com/ВАШ_НИК/frontend-2026`)
6. Пришлите эту ссылку преподавателю

Все изменения и домашние задания выполняйте **в своём fork'е**.

## Установка Node.js и npm

### 1. Скачайте установщик

Перейдите на официальный сайт [https://nodejs.org/](https://nodejs.org/) и скачайте LTS-версию (рекомендуется для большинства пользователей).

### 2. Запустите установщик

Откройте скачанный `.msi` файл и следуйте инструкциям мастера установки:

- Примите лицензионное соглашение
- Выберите путь установки (по умолчанию `C:\Program Files\nodejs\`)
- Убедитесь, что отмечена опция **"Automatically install the necessary tools"** (опционально)
- Нажмите "Install"

### 3. Проверьте установку

Откройте командную строку (`cmd`) или PowerShell и выполните:

```bash
node -v
npm -v
```

Если установка прошла успешно, вы увидите номера версий Node.js и npm.

## Альтернативный способ: установка через NVM для Windows

NVM (Node Version Manager) позволяет легко переключаться между версиями Node.js.

### 1. Установите NVM для Windows

Скачайте установщик из репозитория [coreybutler/nvm-windows](https://github.com/coreybutler/nvm-windows/releases) (файл `nvm-setup.exe`).

### 2. Установите нужную версию Node.js

```bash
nvm install 20.11.0   # установить конкретную версию
nvm use 20.11.0       # переключиться на неё
nvm list              # список установленных версий
```

## Установка Git (опционально, но рекомендуется)

1. Скачайте установщик с [https://git-scm.com/download/win](https://git-scm.com/download/win)
2. Установите с настройками по умолчанию
3. Проверьте:

```bash
git --version
```

## Полезные команды

```bash
# Обновить npm до последней версии
npm install -g npm

# Установить пакеты из package.json
npm install

# Запустить проект
npm run dev

# Собрать проект
npm run build
```

## Устранение неполадок

- Если команды `node` или `npm` не распознаются, перезагрузите компьютер или добавьте путь к Node.js в переменную среды `PATH` вручную.
- Для установки глобальных пакетов без прав администратора настройте префикс npm:

```bash
npm config set prefix "C:\Users\%USERNAME%\AppData\Roaming\npm"
```
