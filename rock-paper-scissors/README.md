# Камень, Ножницы, Бумага

Шаблон для создания игры на JavaScript.

## Файлы проекта

- **`src/game.js`** — открой этот файл и напиши логику игры внутри трёх функций (четвёртая уже готова)
- **`src/game.solution.js`** — готовое решение (если застрял — загляни сюда)
- **`tests/game.test.js`** — тесты, которые проверят правильность твоего кода
- **`package.json`** — настройки проекта

## Что нужно сделать

Открой файл `src/game.js`. Там уже есть три функции, но они пустые. Реализуй их:

1. **`getComputerChoice()`** — компьютер случайно выбирает камень, ножницы или бумагу
2. **`determineWinner(playerChoice, computerChoice)`** — определяет, кто победил
3. **`playRound(playerChoice)`** — проводит один раунд: получает выбор компьютера и возвращает результат

Функция **`playInteractiveGame()`** уже реализована — она запускает полноценную игру в консоли, где ты вводишь свой выбор, а компьютер отвечает случайным.

После каждого изменения запускай тесты командой:

```bash
npm test
```

Когда всё готово, поиграй в консоли:

```bash
npm start
```

Тесты подскажут, что работает, а что ещё нужно доделать.

## Правила игры

- Камень побеждает ножницы
- Ножницы побеждают бумагу
- Бумага побеждает камень
- Одинаковый выбор — ничья
